스프링 시큐리티는 정말 많은 발전이 이루어졌다. 이전에 내가 학습을 했을 때는 5.x 버전이였는데 벌써 마일스톤 기준으로 7버전까지 나온 것을 볼 수 있다. 현재 포스팅부터는 7버전까지는 아니더라도 현재 정식 버전인 6.x버전으로 진행을 해보도록 하겠다.

## 개발환경 & 필요사항

이제 앞으로 포스팅을 하면서 개발 환경과 필요 지식에 대해 미리 소개하려고 한다. 참고하면 좋을 것 같다.

### 개발환경

- JDK17 이상
- Gradle
- PostgreSQL
- 인텔리제이

### 선수지식

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Javascript
